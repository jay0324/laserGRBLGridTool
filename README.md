# LaserGRBL Grid Tool

LaserGRBL Grid Tool is a tool for arrange laserGRBL G code into a table grid G Code.

# Build project

## Install the dependencies

```bash
npm install
```

## run app

```bash
npm run dev
```

## build app

```bash
npm run build
```
